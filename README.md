# HTML Portfolio

## Overview

This repository contains a collection of beginner HTML projects created while learning the fundamentals of web development.
The goal of these projects was to practice building basic web pages using **HTML structure, links, images, and simple layouts**.

These exercises helped me understand how websites are organized and how different pages connect together.

---

## Projects Included

### Movie Ranking Page

A simple webpage that lists favorite movies using HTML headings and paragraphs.

### Birthday Invitation Page

A basic webpage that simulates a birthday invitation including images and event information.

### About Page

A page describing information about the website or author.

### Contact Page

A simple contact information page demonstrating how basic HTML pages can be structured.
